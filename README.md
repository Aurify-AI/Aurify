# 🖥 how to run aurify after fork

1. **clone it**
```sh
git clone <YOUR_FORK_URL>
cd aurify
```

2. **install deps**
```sh
npm install
```

3. **env vars or bust**
make `.env` w ur keys:
```env
VITE_FIREBASE_API_KEY=ur_mom
VITE_GEMINI_API_KEY=ur_dad
```

4. **run the dev server**
```sh
npm run dev
```
open `http://localhost:5173` → bask in ur creation.
